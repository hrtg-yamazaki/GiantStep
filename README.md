# README

## このアプリについて
  
### 概要

生活習慣を記録し、モニタリングするアプリケーションです。睡眠、食事、学習などの習慣に簡単な自己採点を行い、その記録を日々登録することで、累積されたデータをグラフ、カレンダー、表などで閲覧しモニタリングすることが出来ます。自己採点の基準はデフォルトで用意されていますが、各ユーザーが自身で独自に編集することも可能です。

[リンク先](https://giantstep.herokuapp.com/)
